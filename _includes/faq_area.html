Frequently Asked Questions?

Is Geppetto Open Source?

The code and apps generated by Geppetto are Open Source. Geppetto itself uses open source products

How do I get a demo of Geppetto?

Go to our YouTube Channel or Contact us to join our private BETA program

Can Geppetto be run in my Data Center?

Yes - the only requirement to use Geppetto is a Kubernetes cluster

What about Mobile?

Our apps are responsive and we provide Ionic based applications as well

Can you help me get started?

Yes - we offer a quick start program Contact us for the details

What programmimg languages do you support?

By default we use Node and Angular but others are possible
